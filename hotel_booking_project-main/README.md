# Hotel Booking System

A web-based hotel booking system with a clean, coastal-inspired design. Features a responsive user interface, a simulated checkout process, and an administrative panel for tracking transactions.

## Features

- **Modern UI:** Beach-themed aesthetic with a polished layout.
- **Booking Flow:** Easily select dates, rooms, and guests.
- **Simulated Payment:** A secure-looking checkout modal for credit card simulation.
- **Admin Panel:** Track and display transaction details, including payment amounts and masked card information.

## Technologies Used

- HTML5
- Vanilla CSS
- Vanilla JavaScript
